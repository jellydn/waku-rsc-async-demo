## 📋 Requirements

- [bun.sh](https://bun.sh/)

## 📦 Installation

Before installing Waku, ensure you have Bun installed. In your project directory, run:

```sh
bun install
```

## 🔨 Development

To start the development server, execute:

```sh
bun run dev
```

## 🏗 Build

Build your project for production with:

```sh
bun run build
```
